# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-20

### Added

- Core UltraTyped.js library with <2KB gzipped bundle size
- requestAnimationFrame-driven animation engine for 60fps smooth performance
- Zero dependencies - pure vanilla JavaScript implementation
- Pre-tokenized HTML strings for efficient parsing
- Diff-based smart backspacing for optimal performance
- TypeScript type definitions with zero runtime cost
- React adapter with `useUltraTyped` hook
- Vue adapter with `useUltraTyped` composable
- Svelte adapter with `ultratyped` action
- Benchmark system for performance comparison
- Vite-based playground for live demos
- CI/CD workflows for automated builds and releases
- GitHub Pages deployment configuration
